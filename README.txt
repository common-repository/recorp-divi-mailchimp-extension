Integrate Divi Contact Form with MailChimp. Automatically add form submissions to predetermined lists in MailChimp, using its latest API. This MailChimp Extension supports multiple mailing lists and API Keys.

== Description ==
You can see the video documentation [here](https://www.youtube.com/watch?v=m2Av24AObSI)

= Features: =

* Easy to use.
* Single opt-in – no confirmation email send to subscribers.
* Double opt-in – confirmation email send to subscribers.
* GDPR compatible.
* Mailchimp V3 API integrations.
* Use multiple mailing list per contact form (pro feature).

== Installation ==

1. Upload `recorp-divi-mailchimp-extension.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Dashboard > Settings > Mailchimp For Divi.
4. You can mailchimp connection settings on the first tab (Mailchimp) and main form settings on the second tab (Forms). 

You can see the video documentation [here](https://www.youtube.com/watch?v=m2Av24AObSI)

== Frequently Asked Questions ==

= How do I can set new form field for mailchimp merged field =

Go to plugins settings page. If you didn't connect to mailchimp with API yet please connect it first. Then from below select a mailing list and you will see some merged tags. You need to add those tags to the contact form. Add a field and set the "Field Id" as the merged tag. Save the form.
